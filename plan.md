# High Concept and Development Plan

## High Concept

`Make It Out` is a tense, replayable maze-escape game where players navigate procedural spaces under pressure, using movement skill and route decisions to survive and exit.

## Core Pillars

1. Fast readability and movement clarity.
2. Procedural replayability with fair challenge.
3. Strong tension through pacing and limited safety.

## Jam Scope (Current)

- Single playable loop from spawn to exit.
- Procedural maze generation for each run.
- Basic fail/win states and restart flow.
- Minimal UI for state feedback.

## Milestones

## M1 - Playable Vertical Slice

- [ ] Enter maze and move reliably.
- [ ] Reach exit and trigger win.
- [ ] Trigger fail state and restart.

## M2 - Replayability and Tuning

- [ ] Procedural generation pass for variety.
- [ ] Balance pass for difficulty pacing.
- [ ] Add basic run timer and result display.

## M3 - Polish

- [ ] Audio feedback pass.
- [ ] Visual readability pass.
- [ ] Bug fix and performance check pass.

## Risks and Mitigations

- **Scope creep:** freeze feature list after M1 completion.
- **Performance spikes:** profile generation and pooling before polish.
- **Clarity issues:** prioritize readable level layout and UI cues.
